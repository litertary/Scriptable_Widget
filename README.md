## 수정사항 

1. 위젯 배경 모드 
- 컬러 모드 
- 이미지 모드


2. 위젯 개인화 설정 상단 배치
- 설정사항 그룹 배치
- 폰트 및 컬러 설정
- 위젯 상하좌우 패딩 조절
- 날짜 포맷 설정


3. 위젯 레이아웃 모드
- 노말 모드 : 일정 이름 아래 날짜가 있는 모드
- 레프트 모드 : 날짜가 왼쪽에 있는 모드
- 롸이트 모드 : 날짜가 오른쪽에 있는 모드


4. 위젯 사이즈 
- 미디엄
- 라지
